# تطبيق إسلامي شامل - Islamic Mobile App

تطبيق إسلامي شامل يوفر مجموعة من الخدمات الإسلامية الأساسية للمسلمين.

## المميزات الرئيسية

### 📖 القرآن الكريم
- **قائمة السور**: عرض جميع سور القرآن الكريم مع عدد الآيات
- **عرض الآيات**: قراءة آيات كل سورة مع ترقيم الآيات
- **البحث في القرآن**: البحث في نص القرآن الكريم
- **المفضلة**: حفظ الآيات المفضلة وإدارتها
- **التصميم الإسلامي**: واجهة مستخدم جميلة ومريحة للقراءة

### 🕐 أوقات الصلاة
- عرض أوقات الصلاة الخمس
- تذكيرات أوقات الصلاة
- معلومات إضافية عن الصلاة

### 💚 الأذكار
- أذكار الصباح والمساء
- أذكار متنوعة
- عداد للأذكار

### 👥 السنة النبوية
- أحاديث نبوية شريفة
- تصنيف الأحاديث
- البحث في الأحاديث

### 💬 الفتاوى
- فتاوى إسلامية
- تصنيف الفتاوى
- البحث في الفتاوى

### ⚙️ الإعدادات
- تخصيص حجم الخط
- الوضع المظلم
- إعدادات الإشعارات
- معلومات التطبيق

## التقنيات المستخدمة

- **React Native**: إطار العمل الأساسي
- **Expo**: لتطوير التطبيق
- **SQLite**: قاعدة بيانات محلية للقرآن
- **AsyncStorage**: حفظ البيانات المحلية
- **React Navigation**: التنقل بين الشاشات

## هيكل المشروع

```
islamic_app/
├── App.js                 # الملف الرئيسي للتطبيق
├── screens/              # شاشات التطبيق
│   ├── HomeScreen.js     # الشاشة الرئيسية
│   ├── QuranScreen.js    # شاشة القرآن
│   ├── QuranSearchScreen.js # شاشة البحث
│   ├── VersesScreen.js   # شاشة الآيات
│   ├── BookmarksScreen.js # شاشة المفضلة
│   ├── PrayerScreen.js   # شاشة الصلاة
│   ├── AzkarScreen.js    # شاشة الأذكار
│   ├── SunnahScreen.js   # شاشة السنة
│   ├── FatwaScreen.js    # شاشة الفتاوى
│   └── SettingsScreen.js # شاشة الإعدادات
├── services/             # خدمات التطبيق
│   ├── quranDatabase.js  # قاعدة بيانات القرآن
│   ├── bookmarkService.js # خدمة المفضلة
│   └── audioService.js   # خدمة الصوت (مستقبلية)
├── components/           # المكونات المشتركة
│   ├── FontLoader.js     # تحميل الخطوط
│   └── IslamicPatterns.js # الأنماط الإسلامية
└── assets/              # الملفات الثابتة
    ├── data/            # بيانات القرآن
    └── fonts/           # الخطوط العربية
```

## المميزات الجديدة المضافة

- التنقل المباشر للآيات

### 2. نظام المفضلة

- إدارة المفضلة
- عرض تاريخ الإضافة

### 3. تحسينات واجهة المستخدم
- تصميم إسلامي جميل
- أرقام الآيات مع تصميم مميز
- أزرار تفاعلية للمفضلة

### 4. شاشة الإعدادات
- تخصيص حجم الخط
- إعدادات الوضع المظلم
- إعدادات الإشعارات

## كيفية التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn
- Expo CLI

### خطوات التثبيت

1. **تثبيت التبعيات**:
```bash
cd islamic_app
npm install
```

2. **تشغيل التطبيق**:
```bash
npx expo start
```

3. **تشغيل على الجهاز**:
- استخدم تطبيق Expo Go على هاتفك
- امسح رمز QR الذي يظهر في المتصفح
